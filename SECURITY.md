# Security Policy

This is an unofficial, community project maintained on a best-effort basis. It is **not** affiliated
with, endorsed by, or supported by Amazon.

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, report privately using GitHub's
[private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability):
open the repository's **Security** tab → **Report a vulnerability**.

Please include:

- a description of the issue and its impact,
- steps to reproduce (a minimal proof of concept if possible),
- affected version(s), and
- any suggested remediation.

We will acknowledge your report as soon as reasonably possible and work with you on a fix and
coordinated disclosure. Because this is a volunteer-maintained project, response times are best-effort.

## Supported versions

Only the latest released version receives fixes. There are no long-term-support branches.

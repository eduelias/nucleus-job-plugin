# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

In short: be respectful and constructive. Harassment or abusive behavior of any kind is not
tolerated. Report concerns via a private GitHub message to the maintainer or a confidential issue.

The full text is available at
https://www.contributor-covenant.org/version/2/1/code_of_conduct/.
